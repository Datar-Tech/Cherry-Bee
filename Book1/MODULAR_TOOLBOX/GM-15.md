| 機制名稱 | 迷宮素材 (Labyrinth Materials) |
| 機制說明 | 從迷宮魔物屍體中獲得的材料，可用於製作武器、防具和各種高科技產品。是探索者的主要收入來源，也與國力直接相關。 |
| 核心循環 | 擊敗魔物 -> 獲取素材 -> 帶出迷宮 -> 販賣或加工。 |
| 關聯系統 | 迷宮 (GM-01), 探索者 (GM-02), 神匠 (GM-10), 未成年人迷宮產物限制 (GM-17)。 |
| 數值參數 | 稀有度、價值、魔力含量。 |
| 小說依據 | 卡琳用各種魔物素材製作魔法裝備和家具。 |
| 備註 | 迷宮素材的產出量與國力直接相關。 |
