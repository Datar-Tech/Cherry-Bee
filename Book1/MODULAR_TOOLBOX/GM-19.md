| Mechanic_ID | GM-19 |
| 機制名稱 | SC (Super Chat) |
| 機制說明 | 直播平台上的打賞功能，觀眾可以支付高額費用發送醒目留言，以支持實況主。 |
| 核心循環 | 觀眾打賞 -> 實況主獲得收益和關注 -> 留言在直播中突出顯示。 |
| 關聯系統 | 迷宮實況 (GM-03), 頻道訂閱數 (GM-04), 即時觀看人數 (GM-05)。 |
| 數值參數 | 打賞金額。 |
| 小說依據 | 卡琳在收益化後收到巨額SC，引發了「三億之女」的稱號。 |
| 備註 | 高額SC常伴隨「紅SC慶典」，是知名主播頻道上的獨特現象。 |
